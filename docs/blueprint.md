# **App Name**: Himachal Haven

## Core Features:

- Quick Booking Form: A prominently placed form on the homepage allowing users to quickly select check-in/check-out dates and number of guests.
- Interactive Google Map: Embedded Google Map showcasing the resort's location in Himachal, with nearby attractions highlighted.
- Image Gallery: Showcase the resort's nature, rooms, activities, and amenities through a dynamic and visually appealing image gallery.
- WhatsApp Integration: Integrate a WhatsApp chat button for instant communication with resort staff.
- Testimonials Section: Displaying client testimonials and quotes about their stays to boost credibility.
- Online Booking System: A system for processing booking and payments
- AI Powered Travel Guide: An AI-powered tool which provides personalised itineraries and recommendations for local sights.

## Style Guidelines:

- Primary color: Forest green (#347C45) to reflect nature and tranquility.
- Background color: Earthy brown (#E3DAC9) to complement the forest green.
- Accent color: Sky blue (#71A9F7) for highlighting interactive elements.
- Font pairing: 'Alegreya' serif for headings and 'PT Sans' sans-serif for body text, providing both elegance and readability.
- Use custom icons to represent resort highlights such as mountain view, river view, adventure activities, and family-friendly amenities.
- Implement a mobile-first, responsive layout with large visuals and minimal text for an immersive experience.
- Use subtle fade and parallax animations to create smooth transitions and enhance user engagement.
